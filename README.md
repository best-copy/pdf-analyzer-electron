# PDF 컬러/흑백 분석기 (PDF Analyzer)

PDF 파일의 **컬러/흑백 페이지를 자동 분석**하고, 선택한 페이지를 **고품질 흑백으로 변환**해 다시 PDF로 저장하는 Electron 데스크탑 앱입니다. 인쇄 비용 견적서도 자동으로 생성합니다.

## 주요 기능

- **자동 색상 분석** — 각 페이지를 렌더링해 컬러/흑백 여부를 판정 (그레이스케일 비중 ≥ 99.9999%)
- **벡터 레벨 흑백 변환** — 단순 픽셀 변환이 아니라 PDF 내부 색상 연산자·이미지·그라데이션·패턴·별색(Separation/PANTONE)·Indexed 색공간까지 흑백으로 변환 → 프린터가 흑백으로 정확히 인식
- **투명도 보존** — SMask(알파 마스크)를 유지해 투명 배경 로고가 사라지지 않음
- **페이지 편집** — 삭제(되돌리기 지원)·빈 페이지 삽입·90° 회전·드래그로 순서 이동
- **썸네일 미리보기** — 선택 즉시 흑백 미리보기, 다중 선택(Ctrl/Shift+클릭)
- **인쇄 비용 견적서** — 컬러/흑백 단가 기반 자동 견적서 생성 및 PDF 저장
- **빠른 속도** — 페이지 병렬 분석·변환, CPU 코어 수 자동 활용

## 기술 스택

- **Electron** — 데스크탑 앱 셸
- **PDF.js** — PDF 렌더링 및 색상 분석
- **pdf-lib** — PDF 조립·재저장
- **pako** — zlib(Flate) 압축/해제
- Web Worker(`worker-gray.js`) — 흑백 변환 연산을 백그라운드에서 병렬 처리

## 실행 방법

```bash
# 의존성 설치
npm install

# 개발 실행
npm start

# Windows용 포터블 앱 빌드 (dist/win-unpacked 생성)
npm run build
```

> Node.js와 npm이 설치되어 있어야 합니다.

## 프로젝트 구조

```
pdf-analyzer-electron/
├── main.js            # Electron 메인 프로세스 (창 생성, IPC, 줌 단축키)
├── preload.js         # 렌더러에 안전하게 노출되는 API
├── src/
│   ├── index.html     # UI + 분석/변환 로직 (메인 스레드)
│   ├── worker-gray.js # 흑백 변환 Web Worker
│   └── libs/          # pdf.js, pdf-lib, pako (로컬 번들)
└── 작업내역.md         # 상세 작업 이력 (한글)
```

## 단축키

| 단축키 | 동작 | 단축키 | 동작 |
|--------|------|--------|------|
| Ctrl+Z | 삭제 되돌리기 | W | 빈 페이지 추가 |
| Ctrl+[ / ] | 썸네일 축소/확대 | R / L | 오른쪽/왼쪽 회전 |
| Ctrl + / − / 0 | 화면 줌 | B / C | 흑백변환 선택 / 선택 해제 |
| S | 사이드바 접기/펼치기 | | |
